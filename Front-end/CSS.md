# CSS

## 基础

1. 简介

   - 给单元格加上背景色

     - 不使用 css

       ```html
       <table border="1">
         <tr >
             <td bgcolor="gray" >1</td>
             <td bgcolor="gray">2</td>
         </tr>
         
         <tr>
             <td  bgcolor="gray">3</td>
             <td  bgcolor="gray">4</td>
         </tr>
         
         <tr>
             <td  bgcolor="gray">a</td>
             <td  bgcolor="gray">b</td>
         </tr>
         
       </table>
       ```

     - 使用 css

       ```css
       <style>
       td{
         background-color:gray;
       }
       </style>
        
       <table border="1">
         <tr >
             <td>1</td>
             <td>2</td>
         </tr>
         
         <tr>
             <td>3</td>
             <td>4</td>
         </tr>
         
         <tr>
             <td>a</td>
             <td>b</td>
         </tr>
         
       </table>
       ```
